##### Makelangelo #####

http://www.makelangelo.com/

A mural drawing robot and an introduction to 3D printing.

Makelangelo is a polargraph mural drawing robot.  It is intended to teach students 
about programming, physics, trigonometry, and electronics in a fun, goal-oriented 
way.

This computer program that makes running the robot user friendly.
It pairs really well with Makelangelo-firmware, the code in the brain of the robot.
